# Swup Digital Profile

This is a modern, Solana-themed digital profile website designed to showcase social media links with interactive and visually appealing elements. It features a unique blend of green and purple hues, dynamic particle animations, and subtle glow effects to create an engaging user experience.

## Features

*   **Solana-Themed Aesthetics**: A vibrant color palette inspired by the Solana blockchain, featuring striking green (`#14F195`) and purple (`#9945FF`) gradients.
*   **Interactive Particle Background**: A dynamic canvas background with randomly generated particles that react to mouse movement, creating an immersive and ever-changing visual landscape.
*   **Dynamic Glow Effects**: Ambient glow effects subtly illuminate the main content area, enhancing the Solana-inspired theme.
*   **Social Media Integration**: Seamlessly link to various social media profiles (GitHub, X, Telegram, Discord) with custom-designed icons.
*   **Green-Purple Gradient Hover Effects**: Social media icons feature a captivating green text color on hover, combined with a purple glow drop-shadow, creating a distinct interactive experience.
*   **Responsive Design**: Optimized for a flawless viewing experience across all devices, from desktops to mobile phones.
*   **Smooth Animations**: Utilizes Framer Motion for elegant and fluid transitions, making interactions feel natural and polished.

## Technologies Used

*   **React**: A JavaScript library for building user interfaces.
*   **Vite**: A fast build tool and development server for modern web projects.
*   **Tailwind CSS**: A utility-first CSS framework for rapidly styling components.
*   **Framer Motion**: A production-ready motion library for React to power animations.
*   **Lucide React**: A collection of beautiful and customizable open-source icons.
*   **HTML Canvas**: For rendering the interactive particle background.
*   **JavaScript**: The primary programming language for all logic.
